Når man bygger programmet bliver en seedfunktion kaldt som først sletter alle posts og useres og derefter indsætter dummydata.

Når du skal logge ind, skriver du navnet på den bruger du gerne vil ind på.
I vores program seeder vi nogle brugere og dem kan man logge ind som, deres navne ses herunder:
 - Tobias
 - Frederik
 - Morten
 - Klara

Når man er logget ind får en række muligheder får hvad man har lyst til at gøre.
Se feed, user wall, create post, create comment og delete all post. 
Delete all posts er som sådan ikke nogen brugbar funktionalitet, men kan bruges inden der laves et nyt post så det er lettere at finde.

Hvis du vælger comment skal du have et post id at knytte kommentaren til.
Alle posts viser deres ID når de bliver skrevet ud. Så kopier id på den post du vil kommentere og vælg comment, sæt id ind og skriv en kommentar.

Ellers prøv dig frem, det skulle gerne være forholdsvis intuitivt.
